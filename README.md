# 说明 
这是一个基于Trae的测试，工作时间为一个周末。
测试的结果为一个现代化富文本编辑器，支持多种编辑功能和实时协作。

## 功能特性

### 基础编辑
- 文本格式化（粗体、斜体、下划线等）
- 字体大小和颜色调整
- 文本对齐方式
- 缩进控制

### 特殊块支持
- LaTeX 公式编辑
- Mermaid 图表
- 代码块（支持语法高亮）
- 待办事项列表
- 图片插入和编辑

### 列表支持
- 有序列表
- 无序列表
- 待办事项列表
- 支持多级缩进

### 文件操作
- 新建文档
- 打开本地文件
- 保存文件
- 最近文件列表

### 其他功能
- 实时协作支持（基于 Hocuspocus）
- 撤销/重做
- 快捷键支持

## 技术栈

- **前端框架**: React + TypeScript
- **编辑器核心**: Slate.js
- **UI组件**: Radix UI
- **构建工具**: Vite
- **桌面应用框架**: Electron
- **实时协作**: Hocuspocus
